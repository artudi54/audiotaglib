# This is an example file of "fields.csv" file used by accessor-generator.
# The format is as follows:
# TAGTYPE,DATATYPE,TAGNAME,GENERATEMOVE,SETTERTYPE
# "TAGTYPE" corresponds to tag::AudioTag::Type enum values (as string).
# "DATATYPE" is the data type used in tag::AudioTag (ex. std::string, tag::types::Date).
# "TAGNAME" is the tag name written in pascal case (LikeThis).
# "SETTERTYPE" - 0 for primitive, 1 for reference, 2 for reference and move
# Note: treat all fields as a part of AudioTagMap - tag:: prefix in namespace name is unnecessary.
# Example:

String,Artist,std::string,1
Image,ImageIcon,types::Image,2
Date,Date,types::Date,false,1
